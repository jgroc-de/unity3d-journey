## Package de trucs cool

https://drive.google.com/file/d/15WoPHP5ae1bJ6ELG97otNBNoHy3pQzJe/view
Pour l'importer dans unity, faire:
- assets->import package->custom package
- selectionner votre package qui doit avoir ce nom la: UnityCSharpStartKit.unitypackage (probablement dans votre dossier de téléchargement)
- puis appuyer sur import quand il a fini de charger tout

## CheckList des erreurs de scripts

 - il faut que le fichier ait le même nom que la classe
 - pas d'accent dans les scripts dans les noms de classe/variable/fonction 
 - pas d'espace ou de characteres chelou non plus
 - un nom de classe/fonction/variable doit commencer par une lette
 - les fonctions en C# commence par une majuscule (c'est mieux mais pas obligatoire)
 - pour faire un float, ajouter un F à la fin. Ex: 0.1F
